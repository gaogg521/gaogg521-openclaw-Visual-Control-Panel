# 安全披露

若你发现 **gaogg521-openclaw-Visual-Control-Panel** 中与**本仓库代码**相关的安全漏洞（例如未授权访问面板 API、路径穿越、依赖已知 CVE 等），请 **不要** 在公开 Issue 中披露细节。

请通过以下方式私下联系维护者：

- GitHub：向仓库维护者 [@gaogg521](https://github.com/gaogg521) 发送私信说明（若可用），或使用 GitHub **Security advisories**（若仓库已开启）。
- 在邮件/私信中请包含：影响版本或分支、复现步骤、潜在影响范围（**勿**附带可利用的 exploit 以外的攻击性 payload）。

**说明**：与 **OpenClaw 上游**（`openclaw` CLI / Gateway）相关的漏洞，请遵循 [OpenClaw 官方](https://github.com/openclaw/openclaw) 的披露渠道。

我们会在合理时间内确认并回复；修复后会通过 Release / Issue 公开致谢（除非你要求匿名）。
