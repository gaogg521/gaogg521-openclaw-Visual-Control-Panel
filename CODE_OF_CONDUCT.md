# 贡献者公约

## 我们的承诺

为建设开放、友善的环境，我们承诺：参与本项目的每个人都不分年龄、体型、身体健全与否、民族、性征、性别认同与表达、经验、国籍、外貌、种族、宗教或性取向，免受骚扰。

## 我们的标准

有助于积极环境的行为示例：

- 使用友善、包容的语言
- 尊重不同的观点与经验
- 心平气和地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员保持同理心

不可接受的行为示例：

- 性化的语言或意象、不受欢迎的性关注或性暗示
- 挑衅、侮辱/贬损的评论、人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息
- 其他在专业场合中合理认为不恰当的行为

## 我们的责任

维护者有责任阐明可接受行为的标准，并对不当行为采取适当、公平的纠正措施。

维护者有权利和责任删除、编辑与本公约不符的评论、提交、代码、wiki 编辑、issue 等，或暂时、永久禁止任何贡献者认为不当、威胁、冒犯或有害的行为。

## 适用范围

本公约适用于本项目空间内，以及个人在公开场合代表本项目或其社区时。

## 执行

如遇辱骂、骚扰或其他不可接受行为，可通过 GitHub Issue（可标注仅维护者可见的说明）或维护者在 README 中提供的联系方式报告。维护团队将审查并调查全部投诉，并作出被认为必要且适当的回应。维护者有义务对报告者保密。

不遵守本公约的参与者可能被临时或永久限制参与本项目，由维护者决定。
